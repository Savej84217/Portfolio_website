# Savej Salim Shaikh - Portfolio Website

A modern, responsive portfolio website showcasing my skills, projects, and experience as an AIML student.

## Features

- Clean and professional design with smooth animations
- Fully responsive across all devices
- Sections include:
  - Hero section with introduction
  - About section with profile information
  - Projects showcase with detailed descriptions
  - Skills overview with technical and soft skills
  - Education details
  - Contact information

## Technologies Used

- React 18
- Vite
- React Icons
- CSS3 with custom properties
- Modern JavaScript (ES6+)

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```
